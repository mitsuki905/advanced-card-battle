# 要件定義書

## スレスパ風カードバトルゲーム（Web版・DB対応）

***

# 1. 概要

本システムは、Slay the Spire風のターン制カードバトルゲームをWebアプリケーションとして構築し、SQLite3を用いてゲーム状態を管理するものである。  
最小構成で実装し、段階的に機能拡張可能な設計とする。

***

# 2. 動作環境

## 2.1 フロントエンド

* Webブラウザ（Chrome推奨）
* HTML / CSS / JavaScript

## 2.2 バックエンド

* Python 3.x
* フレームワーク：Flask

## 2.3 データベース

* SQLite3

***

# 3. システム構成

```
ブラウザ
   ↓
Flask（API）
   ↓
SQLite3
```

***

# 4. スコープ

## 4.1 含む機能

* ターン制カードバトル
* デッキ／手札／捨て札管理
* エネルギー管理
* マップ進行（ノード選択）
* 戦闘・回復イベント
* DBによる状態保存
* Web UI操作

## 4.2 含まない機能

* 戦闘後カード報酬
* 認証機能
* マルチプレイヤー
* アニメーション演出

***

# 5. ゲーム仕様

***

## 5.1 基本ルール

* プレイヤーと敵がターン制で行動
* HPが0で敗北
* ボス撃破でゲームクリア

***

## 5.2 プレイヤー仕様

* HP：50
* エネルギー：3（ターン開始時に回復）
* ブロック：ターン終了時にリセット

***

## 5.3 敵仕様

### 通常敵

* HP：40

### 強敵

* HP：60

### ボス

* HP：80

***

# 6. カードシステム

***

## 6.1 初期デッキ構成

* Strike ×5
* Defend ×4
* Draw ×1
* Heavy Strike ×1

計11枚

***

## 6.2 カード仕様

### Strike

* コスト：1
* 効果：6ダメージ

### Defend

* コスト：1
* 効果：ブロック +5

### Draw

* コスト：1
* 効果：2枚ドロー

### Heavy Strike

* コスト：2
* 効果：10ダメージ

***

## 6.3 カード使用ルール

* エネルギー不足時は使用不可
* 使用後は捨て札へ移動

***

# 7. デッキ管理

***

## 7.1 構成

* deck（山札）
* hand（手札）
* discard（捨て札）

***

## 7.2 ドロー処理

* 手札が5枚になるまで補充
* 山札が空の場合：
  * 捨て札をシャッフルして再利用

***

# 9. マップ機能

***

## 9.1 ノード種類

* battle（通常敵）
* elite（強敵）
* rest（回復）
* boss（ボス）

***

## 9.2 ノード処理

| 種類     | 内容     |
| ------ | ------ |
| battle | 通常戦闘   |
| elite  | 強敵戦    |
| rest   | HP +15 |
| boss   | ボス戦    |

***

## 9.3 マップ構造

```
通常敵
↓
分岐

└ 強敵
↓
休憩
↓
ボス
```

***

## 9.4 ノード管理

```python
class Node:
```

# 12. データベース設計（SQLite3）

***

## テーブル：game\_state

| カラム           | 内容      |
| ------------- | ------- |
| id            | 主キー     |
| player\_hp    | プレイヤーHP |
| enemy\_hp     | 敵HP     |
| player\_block | ブロック    |
| energy        | エネルギー   |
| deck          | 山札      |
| hand          | 手札      |
| discard       | 捨て札     |
| current\_node | 現在位置    |

***

## 保存形式

* deck / hand / discard：JSON文字列

***

# 13. データ構造

***

## Card

```python
class Card:
    def __init__(self, name, cost, effect):
        self.name = name
        self.cost = cost
        self.effect = effect
```

***

## GameState

```python
class GameState:
    player_hp
    enemy_hp
    player_block
    energy
    deck
    hand
    discard
    current_node
```
