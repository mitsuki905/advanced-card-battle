# ⚔ Card Battle Roguelike (Flask)

Slay the Spire風のターン制カードバトルゲーム。  
Flask + SQLite で構築されたWebアプリケーションです。

---

## 🎮 概要

マップを進みながら敵と戦い、カードを集めてデッキを構築し、  
ボス撃破を目指すローグライクゲームです。

---

## 🚀 起動方法

### 1. セットアップ

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
````

### 2. サーバー起動

```bash
python app.py
```

### 3. ブラウザアクセス

```
http://localhost:5000
```

***

## 🕹 ゲームフロー

```
ゲーム開始
↓
マップ選択
↓
戦闘 / 休憩
↓
報酬選択
↓
次ノードへ進行
↓
ボス撃破 → Act進行
```

***

## ⚔ 戦闘システム

### 基本ルール

| 項目    | 内容         |
| ----- | ---------- |
| エネルギー | 毎ターン3回復    |
| 手札    | 最大5枚ドロー    |
| ブロック  | ターン終了でリセット |
| 勝利条件  | 敵HPを0にする   |

***

### ターンの流れ

1. エネルギー回復（3）
2. カードを5枚ドロー
3. カードを使用
4. ターン終了
5. 敵が行動

***

## 🃏 カードシステム

### 主な効果

* ダメージ
* ブロック
* ドロー
* エネルギー回復
* 状態異常

***

### 状態異常

| 状態         | 効果           |
| ---------- | ------------ |
| Poison     | 毎ターンダメージ（減衰） |
| Weak       | 与ダメージ低下      |
| Vulnerable | 被ダメージ増加      |
| Frail      | ブロック効果低下     |

***

## 🧩 レアリティ

| レア度      | 説明       |
| -------- | -------- |
| Starter  | 初期カード    |
| Common   | 安定性能     |
| Uncommon | 戦略系      |
| Rare     | 高性能・ビルド軸 |

***

## 🎁 報酬システム

* 戦闘勝利後、カード3枚から1枚選択
* スキップ可能

***

## 🗺 マップシステム

分岐型ノード構造を採用しています。

### ノード種類

* 通常戦闘
* エリート
* ボス
* 休憩

***

## 🛌 休憩ノード

以下の行動が選択可能：

* ❤️ 回復（HP +15）
* 🛒 ショップ（カード取得）
* ✂️ カード削除

***

## 🏆 Act（階層システム）

* ボス撃破で次のActへ進行
* Actが進むほど敵が強化されます

### 強化内容

| Act | 内容          |
| --- | ----------- |
| 1   | 基本          |
| 2以降 | 敵HP +20ずつ増加 |

***

## ⚔ 敵システム

* 行動は重み付きランダム
* 行動意図（Intent）が表示されます

***

## 💾 セーブ機能

### 手動セーブ

* 任意の名前で保存可能

### 自動セーブ

* 1スロット管理

### ロード

* セーブデータから復元可能

***

## 🗄 使用技術

* Python（Flask）
* SQLite
* HTML / JavaScript

***

## 🧠 内部設計（概要）

SQLiteでゲームの状態を管理しています：

* プレイヤー（HP / エネルギー / ブロック）
* 敵（HP / Intent / 状態異常）
* カード（デッキ / 手札 / 捨て札）
* 進行状況（マップ / ノード / フェーズ）

***

## 📌 APIエンドポイント

| エンドポイント            | 内容    |
| ------------------ | ----- |
| `/start`           | ゲーム開始 |
| `/state`           | 状態取得  |
| `/action/card`     | カード使用 |
| `/action/end_turn` | ターン終了 |
| `/map/select`      | マップ選択 |
| `/reward/select`   | 報酬選択  |
| `/rest/select`     | 休憩行動  |
| `/manual_save`     | 手動セーブ |
| `/autosave`        | 自動セーブ |
