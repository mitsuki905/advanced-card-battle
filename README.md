# ⚔ カードバトルゲーム

Slay the Spire 風のターン制カードバトルゲームです。  
ブラウザ上でプレイできる Web アプリケーションです。

---

## 🎮 ゲーム概要

マップ上のノードを選択しながら進み、敵を倒してボスを撃破することを目指します。  
手札のカードを使ってダメージを与えたり、ブロックで身を守りながら戦いましょう。

---

## 🚀 起動方法

### 1. 依存パッケージのインストール

```bash
pip install -r requirements.txt
```

### 2. サーバーの起動

```bash
python app.py
```

### 3. ブラウザでアクセス

```
http://localhost:5000
```

---

## 🕹 遊び方

### ゲームの流れ

```
ゲームスタート
↓
マップ画面（次のノードを選択）
↓
戦闘 または 休憩
↓
マップ画面（繰り返し）
↓
ボス戦
↓
ゲームクリア / ゲームオーバー
```

### 戦闘の流れ

1. ターン開始時にエネルギーが **3** 回復し、手札が **5枚** になるまでドローします
2. 手札のカードをクリックして使用します（エネルギーが足りないカードは使用不可）
3. 「**ターン終了**」ボタンを押すと敵が攻撃します
4. 敵の HP を 0 にすると戦闘勝利です

---

## 🗺 マップ構造

```
通常敵（n0）
↓
分岐
├ 通常敵（n1a）
└ 強敵（n1b）
↓
休憩（n2）
↓
ボス（n3）
```

| ノード種類 | 内容 |
|-----------|------|
| 通常敵    | 通常の戦闘（敵 HP: 40） |
| 強敵      | 強い敵との戦闘（敵 HP: 60） |
| 休憩      | HP +15 回復 |
| ボス      | ボス戦（敵 HP: 80）。撃破でゲームクリア |

---

## 🃏 カード一覧

| カード名 | コスト | 効果 |
|---------|--------|------|
| Strike | 1 | 6 ダメージ |
| Defend | 1 | ブロック +5 |
| Draw | 1 | 2 枚ドロー |
| Heavy Strike | 2 | 10 ダメージ |

### 初期デッキ構成（計 11 枚）

- Strike × 5
- Defend × 4
- Draw × 1
- Heavy Strike × 1

---

## 👤 プレイヤー仕様

| 項目 | 値 |
|------|----|
| HP | 50 |
| エネルギー | 3（ターン開始時に回復） |
| ブロック | ターン終了時にリセット |

---

## ⚔ 敵仕様

| 種類 | HP | 攻撃力 |
|------|----|--------|
| 通常敵 | 40 | 6 ダメージ / ターン |
| 強敵 | 60 | 6 ダメージ / ターン |
| ボス | 80 | 6 ダメージ / ターン |

---

## 📋 ゲームルール

- プレイヤーの HP が 0 になると **ゲームオーバー**
- ボスを撃破すると **ゲームクリア**
- ブロックは敵の攻撃ダメージを軽減します（ターン終了時にリセット）
- 山札が空になると、捨て札をシャッフルして再利用します
- 戦闘中は敵を倒すまでマップへ進めません
