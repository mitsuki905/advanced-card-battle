
# ⚔ カードバトルゲーム

Slay the Spire 風のターン制カードバトルゲームです。  
ブラウザ上でプレイできる Web アプリケーションです。

---

## 🎮 ゲーム概要

マップ上のノードを進みながら敵と戦い、ボスを撃破しつつ  
より強いデッキを構築していくローグライクゲームです。

✅ カードデッキ構築  
✅ 状態異常（毒・弱体）  
✅ 削除・最適化  
✅ Act（階層）進行による難易度上昇  

---

## 🚀 起動方法

### 1. セットアップ（初回のみ）

```

run\_setup.bat をダブルクリック

````

または手動で：

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
````

***

### 2. サーバーの起動

```
run_app.bat をダブルクリック
```

または手動で：

```bash
venv\Scripts\activate
python app.py
```

***

### 3. ブラウザでアクセス

```
http://localhost:5000
```

***

## 🕹 遊び方

### ゲームの流れ

```
ゲーム開始
↓
マップ選択
↓
戦闘 or 休憩
↓
戦闘勝利 → 報酬選択
↓
マップ進行
↓
ボス撃破 → 次のActへ
↓
繰り返し
```

***

## 🏆 Act（階層システム）

* ボスを倒すと **次の Act に進行**
* Actが上がると敵が強くなる

| Act | 敵強化     |
| --- | ------- |
| 1   | 基本ステータス |
| 2   | HP +20  |
| 3   | HP +40  |

***

## ⚔ 戦闘の流れ

1. ターン開始時にエネルギーが **3** 回復
2. 手札を **5枚までドロー**
3. カードを使用して攻撃・防御・補助
4. ターン終了で敵行動
5. 敵HPが0で勝利

***

## 🧠 カードシステム

### 基本効果

* ダメージ
* ブロック
* ドロー
* 毒（継続ダメージ）
* 弱体（ダメージ減少）
* エネルギー増加

***

## 🃏 カード種類

### 初期デッキ（11枚）

* Strike ×5
* Defend ×4
* Draw ×1
* Heavy Strike ×1

***

### 報酬カード（拡張）

* Bash
* Iron Wave
* Shrug It Off
* Pommel Strike
* Twin Strike
* Armaments
* Poison Strike

***

## 🎁 報酬システム

戦闘勝利後：

* 3枚のカードが提示される
* 1枚選択可能 or スキップ可能

***

## 🔥 カード削除

休憩ノードで実行可能

* デッキのカードを1枚削除できる
* デッキ圧縮で戦略強化

***

## 🗺 マップ構造

```
通常敵
↓
分岐
├ 通常
└ 強敵
↓
休憩（回復 / 削除）
↓
ボス
```

***

## 💾 セーブ機能

### ✅ 手動セーブ

* 任意のスロットに保存

### ✅ 自動セーブ

* ページ離脱時に自動保存

### ✅ ロード

* 自動 / 手動どちらも可能

***

## 👤 プレイヤー仕様

| 項目    | 値          |
| ----- | ---------- |
| HP    | 50         |
| エネルギー | 3          |
| ブロック  | ターン終了でリセット |

***

## ⚔ 敵仕様

| 種類  | HP  | 特徴       |
| --- | --- | -------- |
| 通常敵 | 40  | 基本       |
| 強敵  | 60  | やや強い     |
| ボス  | 80+ | Actごとに強化 |

***

## 📋 ゲームルール

* HP 0 → ゲームオーバー
* ボス撃破 → 次Actへ
* デッキはシャッフルで再利用
* 戦闘中は移動不可

***
