# ⚔ Card Battle Roguelike (Flask)

Slay the Spire風のターン制カードバトルゲーム。  
Flask + SQLite を使用したブラウザ向けWebアプリケーションです。

---

## 🎮 概要

マップを進みながら敵と戦い、カードを集めてデッキを構築し、  
ボス撃破を目指すローグライクゲームです。

---

## 🚀 起動方法

### 1. セットアップ

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
````

### 2. サーバー起動

```bash
python app.py
```

### 3. ブラウザアクセス

```
http://localhost:5000
```

***

## 🕹 ゲームの流れ

```
ゲーム開始
  ↓
マップ選択
  ↓
戦闘 / 休憩
  ↓
報酬選択
  ↓
次ノードへ進行
  ↓
ボス撃破 → Act進行
```

***

## ⚔ 戦闘システム

### 基本ルール

| 項目    | 内容         |
| ----- | ---------- |
| エネルギー | 毎ターン 3 回復  |
| 手札    | 最大 5 枚ドロー  |
| ブロック  | ターン終了でリセット |
| 勝利条件  | 敵HPを 0 にする |

***

### ターンの流れ

1. エネルギー回復（+3）
2. カードを 5 枚ドロー
3. カードを使用
4. ターン終了
5. 敵が行動

***

## 🃏 カードシステム

### 主な効果

* ダメージ
* ブロック
* ドロー
* エネルギー回復
* 状態異常

***

### 状態異常

| 状態         | 効果                 |
| ---------- | ------------------ |
| Poison     | 毎ターンダメージ（ターンごとに減少） |
| Weak       | 与ダメージ低下            |
| Vulnerable | 被ダメージ増加            |
| Frail      | ブロック効果低下           |

***

## 🧩 レアリティ

| レア度      | 特徴          |
| -------- | ----------- |
| Starter  | 初期カード       |
| Common   | 安定性能        |
| Uncommon | 戦略の幅を広げる    |
| Rare     | 強力な効果・ビルドの軸 |

***

## 🎁 報酬システム

* 戦闘勝利後、カード 3 枚から 1 枚選択
* スキップ可能

***

## 🗺 マップシステム

分岐型ノード構造を採用しています。

### ノード種類

* ⚔ 通常戦闘
* 💀 エリート
* 👑 ボス
* 🛌 休憩

***

## 🛌 休憩ノード

以下の行動から選択できます：

* ❤️ 回復（HP +15）
* 🛒 ショップ（カード取得）
* ✂️ カード削除

***

## 🏆 Act（階層システム）

* ボス撃破で次の Act に進行
* Act が進むほど敵が強化されます

### 敵強化

| Act  | 内容          |
| ---- | ----------- |
| 1    | 基本ステータス     |
| 2 以降 | 敵HP +20ずつ増加 |

***

## ⚔ 敵システム

* 行動は重み付きランダムで決定
* 行動意図（Intent）が表示されます

***

## 💾 セーブ機能

### 手動セーブ

* 任意の名前で保存可能

### 自動セーブ

* 1スロットで管理

### ロード

* セーブデータから復元可能

***

## 🗄 使用技術

* Python（Flask）
* SQLite
* HTML / JavaScript

***

## 📌 APIエンドポイント

| エンドポイント            | 内容    |
| ------------------ | ----- |
| `/start`           | ゲーム開始 |
| `/state`           | 状態取得  |
| `/action/card`     | カード使用 |
| `/action/end_turn` | ターン終了 |
| `/map/select`      | マップ選択 |
| `/reward/select`   | 報酬選択  |
| `/rest/select`     | 休憩行動  |
| `/manual_save`     | 手動セーブ |
| `/autosave`        | 自動セーブ |
