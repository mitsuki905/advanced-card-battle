# ⚔ カードバトルゲーム

Slay the Spire 風のターン制カードバトルゲームです。  
ブラウザ上でプレイできる Web アプリケーションです。

---

## 🎮 ゲーム概要

マップ上のノードを選択しながら進み、敵を倒してボスを撃破することを目指します。  
手札のカードを使ってダメージを与えたり、ブロックで身を守りながら戦いましょう。  
戦闘勝利後はカード報酬を獲得してデッキを強化できます。

---

## 🚀 起動方法

### 1. セットアップ（初回のみ）

```
run_setup.bat をダブルクリック
```

または手動で：

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

### 2. サーバーの起動

```
run_app.bat をダブルクリック
```

または手動で：

```bash
venv\Scripts\activate
python app.py
```

### 3. ブラウザでアクセス

```
http://localhost:5000
```

## 🕹 遊び方

### ゲームの流れ

```
ゲームスタート（または自動セーブ/手動セーブからロード）
↓
マップ画面（次のノードを選択）
↓
戦闘 または 休憩
↓
戦闘勝利 → カード報酬選択（デッキ構築）
↓
マップ画面（繰り返し）
↓
ボス戦
↓
ゲームクリア / ゲームオーバー
```

### 戦闘の流れ

1. ターン開始時にエネルギーが **3** 回復し、手札が **5枚** になるまでドローします
2. 手札のカードをクリックして使用します（エネルギーが足りないカードは使用不可）
3. 「**ターン終了**」ボタンを押すと敵が攻撃します
4. 敵の HP を 0 にすると戦闘勝利 → カード報酬画面へ

### カード報酬

- 戦闘勝利後、3枚のカード候補が表示されます
- 1枚選択するとデッキに追加されます
- 「スキップ」でカードを取らずにマップへ戻ることもできます

## 🗺 マップ構造

```
通常敵（n0）
↓
分岐
├ 通常敵（n1a）
└ 強敵（n1b）
↓
休憩（n2）
↓
ボス（n3）
```

| ノード種類 | 内容 |
|-----------|------|
| 通常敵    | 通常の戦闘（敵 HP: 40） |
| 強敵      | 強い敵との戦闘（敵 HP: 60） |
| 休憩      | HP +15 回復 |
| ボス      | ボス戦（敵 HP: 80）。撃破でゲームクリア |

## 💾 セーブ / ロード機能

### 手動セーブ
- ゲーム中の「💾 セーブ」ボタンからモーダルを開く
- セーブ名を入力して「このデータをセーブ」をクリック
- 複数のセーブデータを保存可能

### 自動セーブ
- ブラウザのページを閉じる・離脱する際に自動的に保存
- 最新の1件のみ保持（上書き）

### ロード
- スタート画面の「⏱ 自動セーブから再開」ボタン
- 「💾 セーブデータを選択」からモーダルを開いてロード

---

## 📋 ゲームルール

- プレイヤーの HP が 0 になると **ゲームオーバー**
- ボスを撃破すると **ゲームクリア**
- ブロックは敵の攻撃ダメージを軽減します（ターン終了時にリセット）
- 山札が空になると、捨て札をシャッフルして再利用します
- 戦闘中は敵を倒すまでマップへ進めません
- 報酬フェーズ中はマップへ進めません（カード選択またはスキップが必要）
